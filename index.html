Intro

Hello :D
I'm chococat (or I go by that here, at least). I'm currently still setting up the site! You'll see a lot of unfinished decorations lying around.. they'll be there for a while.

This website is my little journal for self-expression. You'll find bloggies, media I like, perhaps some coding projects and some photography. I really like the atmosphere of DIY on the indie web. When you visit a website, it's like being invited to someone's house. ^v^

I started to explore the world of web development during the summer of 2020. So, instead of repeatedly dying from OP players in florr.io, I played flexbox frog. Then I made small web projects on and off for a while. I've since learned a lot more about web development, like modern full-stack frameworks like React, reasons why Javascript sucks, and how some back-end libraries work; I taught myself a bit of UI design, etc. I'm kind of a full-stack developer! Maybe like, a half-baked one.

I'm familiar with a couple programming languages too. It's always really fun to create stuff, especially interactive and visual programs. 

Hope you stay a while and enjoy browsing :)
